Job Apply Assistant — COMECE AQUI (macOS)
==========================================

Siga estes 3 passos na ordem:

1) EXTRAIR
   - Dê duplo clique no arquivo .zip baixado

   Pelo Terminal (substitua o caminho do .zip):
   cd ~/Downloads
   unzip JobApplyAssistant-*-mac-arm64.zip -d JobApplyAssistant
   cd JobApplyAssistant

2) DESBLOQUEAR (importante na primeira vez)
   - Dê duplo clique em: Desbloquear arquivos.command
   - Se o Mac perguntar, confirme que deseja abrir

   Pelo Terminal (na pasta extraída — copie linha por linha):
   xattr -dr com.apple.quarantine .

3) INSTALAR
   - Dê duplo clique em: Install JobApply Assistant.command
   - O app vai para Aplicativos e um atalho aparece na Área de Trabalho

   Pelo Terminal (na pasta extraída — copie linha por linha):
   xattr -dr com.apple.quarantine JobApplyAssistant.app
   cp -R JobApplyAssistant.app /Applications/
   open /Applications/JobApplyAssistant.app

PERFIL JÁ PREENCHIDO (edição Guilherme)
---------------------------------------
Na primeira abertura, o app instala automaticamente:
  - Nome, e-mail, telefone, LinkedIn
  - Currículo (default_resume.docx)
  - Arquivos de experiência para a IA
  - Chave Groq (se incluída na build)

Você só precisa fazer uma vez:
  1. Settings → Platform Login → LinkedIn (e-mail/senha, não Google)
  2. Settings → Import Jobs from Home Server:
       URL:  https://jobs.krassusky.com
       Token: peça o SYNC TOKEN ao Luis (não é a senha do site)
       → Test connection
       → Import pending jobs
       → Pull shared profile & search
  3. Candidaturas → Generate Resume → Apply → Approve

IMPORTANTE — duas credenciais diferentes:
  - Mac app (campo Sync token) = token de sync (Luis envia)
  - Site https://jobs.krassusky.com/dashboard = login guilherme
    (senha enviada à parte pelo Luis — NÃO use no campo Sync token)

Settings em cinza? Clique Edit para alterar; sync continua usável.
Durante import/generate/apply aparece um overlay de carregamento.

Abas neste Mac: Candidaturas, Experiência, Dados, Configurações
(Painel/busca fica no servidor de casa)

Se o Mac bloquear na primeira abertura:
   - Clique com o botão direito em JobApplyAssistant.app → Abrir → Abrir
   - (só precisa fazer isso uma vez)

Não precisa do Google Chrome — no Mac usamos o motor Safari integrado.

Download atualizado (Guilherme — perfil pré-preenchido):
