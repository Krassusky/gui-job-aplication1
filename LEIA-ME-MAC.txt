JobApply Assistant — Guia rápido (macOS)
========================================

DOWNLOAD
--------
Baixe a versão **1.0.12** (Guilherme — perfil pré-preenchido) em:
https://github.com/Krassusky/gui-job-aplication1/releases/tag/v1.0.12

Já instalado? Configurações → Updates → Check for updates → Update now
(dados em ~/.autoapply/ não são apagados; o app reinicia sozinho e cria atalho na Área de Trabalho)

Guia completo com comandos do Terminal (obrigatório se o duplo clique falhar):
  INSTALACAO-MAC-GUILHERME.md (vem dentro do .zip)

No Mac com chip Apple (M1/M2/M3/M4):
  JobApplyAssistant-X.Y.Z-mac-arm64.zip

No Mac Intel mais antigo:
  JobApplyAssistant-X.Y.Z-mac-x64.zip

INSTALAÇÃO (3 passos)
---------------------
1. Extraia o .zip (duplo clique)
2. Dê duplo clique em Desbloquear arquivos.command
   (remove o bloqueio de segurança do download)
3. Dê duplo clique em Install JobApply Assistant.command
   (copia para Aplicativos e cria atalho na Área de Trabalho)

Se o Mac ainda bloquear na primeira abertura:
   Botão direito em JobApplyAssistant.app → Abrir → Abrir

Instale o Google Chrome (obrigatório) se ainda não tiver.

PRIMEIRO USO (edição Guilherme — perfil pré-preenchido)
-------------------------------------------------------
Na primeira abertura o app instala automaticamente perfil, currículo,
experiências e chave Groq (quando a build inclui o preset).

Você só precisa:
1. Settings → Platform Login → LinkedIn (e-mail/senha — não use Google)
2. Settings → Import Jobs from Home Server
   - URL: https://jobs.krassusky.com
   - Sync token: peça ao Luis (não é a senha do dashboard)
   - Test → Import pending jobs → Pull shared profile & search
3. Applications → revisar vagas importadas

Dashboard web (opcional): https://jobs.krassusky.com/dashboard
Login guilherme + senha (Luis envia à parte) — NÃO use no campo Sync token.

Se instalou uma build genérica (sem preset), use Configurar Guilherme.command
no pacote de setup ou peça o zip guilherme-setup.

PRIMEIRO USO (build genérica)
-----------------------------
1. Siga o assistente de configuração (perfil, currículo PDF, LinkedIn)
2. Use a aba "Guia" para os passos na ordem
3. No Painel, clique Iniciar e revise cada vaga antes de candidatar

ATUALIZAR
---------
Baixe a nova versão no GitHub (link acima) e substitua o JobApplyAssistant.app
na pasta Aplicativos.

Seus dados ficam em: /Users/SEU_NOME/.autoapply/
